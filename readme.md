<h2>Mod Description</h2>
<p style="text-align:center"><a href="https://serilum.com/" target="_blank" rel="nofollow"><img src="https://github.com/Serilum/.cdn/raw/main/description/header/header.png" alt="" width="838" height="400"></a></p>
<p style="text-align:center"><a href="https://curseforge.com/members/serilum/projects" target="_blank" rel="nofollow"><img src="https://raw.githubusercontent.com/Serilum/.data-workflow/main/badges/svg/curseforge.svg" width="200"></a> <a href="https://modrinth.com/user/Serilum" target="_blank" rel="nofollow"><img src="https://raw.githubusercontent.com/Serilum/.data-workflow/main/badges/svg/modrinth.svg" width="200"></a> <a href="https://patreon.com/serilum" target="_blank" rel="nofollow"><img src="https://raw.githubusercontent.com/Serilum/.data-workflow/main/badges/svg/patreon.svg" width="200"></a> <a href="https://youtube.com/@serilum" target="_blank" rel="nofollow"><img src="https://raw.githubusercontent.com/Serilum/.data-workflow/main/badges/svg/youtube.svg" width="200"></a></p>
<p><strong><span style="font-size:24px">Requires the library mod <a style="font-size:24px" href="https://curseforge.com/minecraft/mc-mods/collective" target="_blank" rel="nofollow">Collective</a>.</span></strong><br><br><strong>&nbsp;&nbsp;&nbsp;This mod is part of <span style="color:#008000"><a style="color:#008000" href="https://curseforge.com/minecraft/modpacks/the-vanilla-experience" target="_blank" rel="nofollow">The Vanilla Experience</a></span> modpack and <span style="color:#666"><a style="color:#666" href="https://curseforge.com/minecraft/mc-mods/serilums-customization-bundle" target="_blank" rel="nofollow">Serilum's Customization Bundle</a></span> mod.</strong><br><span style="font-size:18px">Save and Load Inventories adds commands which allow the user to easily save their current inventory to a file, load an inventory from a file and list the current saved inventories. Useful for administrators or event servers. NBT tags are supported, any things like written books or enchantments won't be a problem. Players need cheat access to use the commands.<br><br></span><span style="font-size:18px">Th</span><span style="font-size:18px">e inventories are saved in ./config/saveandloadinventories/*.txt.</span></p>
<p><br><span style="font-size:18px"><strong>Commands:</strong></span><br><span style="font-size:14px"><em>/saveinventory &lt;inventory_name&gt;</em> - Saves the current inventory of the user as &lt;inventory_name&gt;.</span><br>&nbsp; alias: <em>/si &lt;inventory_name&gt;</em><br><span style="font-size:14px"><em>/loadinventory &lt;inventory_name&gt;</em> - Loads an inventory named &lt;inventory_name&gt; previously saved.</span><br>&nbsp; alias: <em>/li &lt;inventory_name&gt;</em><br><span style="font-size:14px"><em>/loadinventory &lt;inventory_name&gt; &lt;player_name&gt;&nbsp;</em>- Loads an inventory to the target player named &lt;player_name&gt;<em><br><span style="font-size:12px">&nbsp; alias: /li &lt;inventory_name&gt; &lt;player_name&gt;</span><br>/listinventories</em> - Lists all saved inventories.<br><br></span><span style="font-size:18px"><strong>Examples:</strong></span><br><span style="font-size:14px">Save the current inventory with the name 'diamond' (<em>/saveinventory diamond</em>):</span><br><picture><img src="https://github.com/Serilum/.cdn/raw/main/projects/save-and-load-inventories/a.jpg" width="1142" height="591"></picture><br><br><span style="font-size:14px">And when loading it (<em>/loadinventory diamond</em>):</span></p>
