<p><span style="font-size:18px">Save and Load Inventories adds commands which allow the user to easily save their current inventory to a file, load an inventory from a file and list the current saved inventories. Useful for administrators or event servers. NBT tags are supported, any things like written books or enchantments won't be a problem. Players need cheat access to use the commands.<br><br></span><span style="font-size:18px">Th</span><span style="font-size:18px">e inventories are saved in ./config/saveandloadinventories/*.txt.</span></p>
<p><br><span style="font-size:18px"><strong>Commands:</strong></span><br><span style="font-size:14px"><em>/saveinventory &lt;inventory_name&gt;</em> - Saves the current inventory of the user as &lt;inventory_name&gt;.</span><br>&nbsp; alias: <em>/si &lt;inventory_name&gt;</em><br><span style="font-size:14px"><em>/loadinventory &lt;inventory_name&gt;</em> - Loads an inventory named &lt;inventory_name&gt; previously saved.</span><br>&nbsp; alias: <em>/li &lt;inventory_name&gt;</em><br><span style="font-size:14px"><em>/loadinventory &lt;inventory_name&gt; &lt;player_name&gt;&nbsp;</em>- Loads an inventory to the target player named &lt;player_name&gt;<em><br><span style="font-size:12px">&nbsp; alias: /li &lt;inventory_name&gt; &lt;player_name&gt;</span><br>/listinventories</em> - Lists all saved inventories.<br><br></span><span style="font-size:18px"><strong>Examples:</strong></span><br><span style="font-size:14px">Save the current inventory with the name 'diamond' (<em>/saveinventory diamond</em>):</span><br><img src="https://github.com/ricksouth/serilum-mc-mods/raw/master/cdn/save-and-load-inventories/a.jpg" width="1142" height="591"><br><br><span style="font-size:14px">And when loading it (<em>/loadinventory diamond</em>):</span></p>
